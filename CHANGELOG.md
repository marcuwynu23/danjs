## [1.0.2] - 2024-XX-XX

### Added
- Initial release of DAN parser and encoder
- Support for key-value pairs with type inference
- Nested block syntax with `{}`
- Array support with `[]`
- Native table support with `table()` syntax
- Comment support (`#` and `//`)
- Buffer input support for `decode()`
- Empty file handling
- Full TypeScript support with type definitions
- Comprehensive test suite
- Examples and documentation

### Features
- Human-readable syntax
- Automatic type inference (numbers, booleans, strings)
- Single-pass parsing for performance
- Pretty printing in encoder
- Table auto-detection in encoder
